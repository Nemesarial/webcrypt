About WebCrypt

WebCrypt is open source in browser encryption application.

WebCrypt fully secure way to transfer sensitive data, as no messages stored on the server and to encrypt nothing is transmitted to the server, the whole encryption process is happening in your browser.

WebCrypt is open and licensed under the GNU GPL. It is based on great Stanford Javascript Crypto Library and hosted on GitHub Pages.
